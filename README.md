# 🛒 E-Commerce Frontend Project

នេះគឺជាគម្រោងគេហទំព័រលក់ទំនិញអនឡាញ (E-commerce) ដែលរៀបចំឡើងដោយប្រើប្រាស់ **React.js**។ វាមានមុខងារពេញលេញចាប់ពីការមើលទំនិញ រហូតដល់ការកុម្ម៉ង់ទិញ។

## 🚀 លក្ខណៈពិសេស (Features)
* **Product Catalog:** បង្ហាញទំនិញយ៉ាងស្អាតតាមរយៈ Product Cards។
* **Shopping Cart:** អាចថែមទំនិញ ដកទំនិញ និងគណនាតម្លៃសរុប (Context API)។
* **User Authentication:** ប្រព័ន្ធ Login និង Protected Routes សម្រាប់ការពារទំព័រ Checkout និង Order។
* **Theme Management:** អាចដូរ Mode ងងឹត ឬភ្លឺ (Dark/Light Mode)។
* **Responsive Design:** ដំណើរការបានយ៉ាងល្អលើទូរស័ព្ទ និងកុំព្យូទ័រ។

## 🛠️ បច្ចេកវិទ្យាដែលប្រើ (Tech Stack)
* **Frontend:** React.js, Tailwind CSS (ឬ CSS Modules)
* **State Management:** React Context API
* **Routing:** React Router DOM
* **Icons:** Lucide React / FontAwesome

## 📂 រចនាសម្ព័ន្ធ Folder (Project Structure)
* `components/`: UI components ដែលអាចប្រើឡើងវិញបាន (Navbar, Footer, etc.)
* `contexts/`: គ្រប់គ្រង Global State (Cart, User, Theme)
* `pages/`: ទំព័រនីមួយៗនៃ Website (Home, Cart, Product Details)
* `data/`: ទិន្នន័យគំរូសម្រាប់បង្ហាញ (Mock Data)
